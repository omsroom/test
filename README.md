# AI体験動画教材パッケージ LP

AI教育動画教材パッケージのランディングページを、保守しやすい静的HTML/CSS/JavaScriptとして再構築したものです。

## 公開URL

https://rawcdn.githack.com/omsroom/test/b289d0b341e48b686b4c1fab0567a0a06dab54d5/index.html

## ファイル

- `index.html`：ページ構造・SEOメタ情報
- `styles.css`：レスポンシブデザイン
- `script.js`：モバイルメニュー、スクロール演出
- `.nojekyll`：静的ホスティング設定

## 主な対応

- PC／タブレット／スマートフォン対応
- 資料請求・導入相談へのCTA
- セマンティックHTMLとキーボード操作
- OGP・description設定
- `prefers-reduced-motion`対応
